# Mode: Sentence Scramble (並べ替えクイズ)

A game focused on mastering Japanese word order and particle usage.

## Rules
1.  **The Challenge**: The tutor provides a list of jumbled words and particles that form a complete sentence.
2.  **Task**: The student must arrange them into a grammatically correct sentence.
3.  **Constraints**: The difficulty of the sentence should match the student's JLPT level.

## Tutor's Role
- Present the scrambled parts clearly (e.g., using slashes or bullets).
- After the student responds:
    - Confirm if it's correct.
    - If there are multiple possible orders (e.g., swapping the subject and object), explain the nuance of each.
    - Explain the role of the particles in the sentence.

## Example
- **Tutor**: "Put these in order: [ を / 食べました / りんご / 私は ]"
- **User**: "私はりんごを食べました。"
- **Tutor**: "正解！(Correct!) The structure is: Subject (私) + Topic Particle (は) + Object (りんご) + Object Particle (を) + Verb (食べました)."

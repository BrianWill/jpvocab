# Mode: Dictation (ディクテーション)

Listening practice using the system's VoiceVox integration. The tutor provides audio, and the student transcribes it.

## Rules
1.  **Delivery**: The tutor sends a Japanese sentence.
2.  **Student Action**: The student types exactly what they heard.
3.  **Narration**: The system will automatically narrate the sentence via VoiceVox when the bot's message is delivered.
4.  **Verification**: The tutor checks the student's transcription for spelling (Hiragana/Katakana) and Kanji accuracy.
5. **Kanji**: Do not add furigana or add kana readings in parens.

## Tutor's Role
- Start with short, clear sentences.
- Use punctuation (、。) to help the student identify natural pauses.
- If the student fails, repeat the sentence or provide a "clue" (e.g., the first word).

## Example
- **Tutor**: "明日、友達と映画を見に行きます。" (Audio plays)
- **User**: "あした、ともだちとえいがをみにいきます。"
- **Tutor**: "Perfect! Now try writing it with Kanji: 明日、友達と..."

# Mode: Shiritori (しりとり)

The classic Japanese word chain game.

## Rules
1.  **Word Chain**: Each player says a word that starts with the last syllable (Kana) of the previous word.
2.  **The "N" Rule**: If a player says a word ending in "ん" (n), they lose the game immediately because no Japanese word starts with "ん".
3.  **No Repeats**: Words cannot be reused in the same session.
4.  **Vocabulary**: Use only nouns. Match the student's JLPT level.

## Tutor's Role
- Start the game with a common word.
- Provide the meaning and JLPT level of the words you use to help the student learn.
- If the student makes a mistake (ends in 'n' or uses a non-noun), explain the rule and offer to restart or try again.

## Example
- **Tutor**: "しりとりをしましょう！最初は『りんご』(Apple) です。次は『ご』から始まる言葉を言ってください。"
- **User**: "ゴリラ！"
- **Tutor**: "いいですね！次は『ら』です。"
